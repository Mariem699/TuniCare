# TuniCare - Site Web de Gestion des Patients

## Description
TuniCare est un site web conçu pour faciliter la gestion des patients par les médecins. Elle permet aux patients de suivre leurs données de santé, de prendre des rendez-vous, et aux médecins de gérer leurs consultations et interventions.

## Fonctionnalités
- Création et gestion de compte patient/médecin.
- Suivi des mesures de santé (poids, glucose, tension, eau).
- Prise de rendez-vous en ligne avec sélection de créneaux.
- Gestion du planning du médecin avec événements et rendez-vous confirmés.
- Notifications de rappels (médicaments, rendez-vous).
- Interaction directe entre le patient et le médecin

## Prérequis
- PHP 
- MySQL
- Serveur web (Apache, via XAMPP)
- HTML & CSS & JavaScript

## Installation
1. visit: https://github.com/Mariem699/TuniCare
1. Clone le repository 
2. Crée une base de données MySQL (nommée par exemple `tunicare`).
3. Importer la base de données.
4. Lance ton serveur web localement.

## Utilisation
- Connecte-toi avec ton compte (patient ou médecin).
- Pour les patients : tu peux consulter tes données de santé, prendre des rendez-vous, et suivre tes objectifs.
- Pour les médecins : visualise ton agenda, ajoute des événements et gère les consultations.
